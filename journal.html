journal entry 2

the web is essentially a bunch of connected computers that have websites. when a computer's connected to the web and is spitting out a website it's considered a server because it's serving the website.
everything in html has to have a start and end tag. all the code has to be within the "html" tags for it to run. it also largely relies on the head and body sections, which provide functions for the code and actual actions for it respectively.
overall, coding has made me feel incredbily frustrated. this is especially true when it comes to the person card and the website, as i feel we were just thrown into this right off the bat.
for a paragraph you use a "p" tag, for which there is no end tag just as for the break tag which is "br". for headings you use "h" tags ranging from "h1" to "h6" decreasing in size as the numbers increase.
in order to italicize, an "em" tag is used on either side of the text, one being an opening tag and the other being a closing tag. the "strong" tag is used to bold text, and is also located on either side of the text, one being an opening tag and the other being a closing tag. finally, if you want to bold and italicize at the same time you can put one inside of the other, wrapping the entire text with one and the specific section with both with the other tag directly around it.
